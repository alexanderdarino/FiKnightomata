Contains classes that represent finite automata and their execution on a given input. Executing a {@link finiteautomaton.FiniteAutomaton} on a given input returns an {@link finiteautomaton.Execution}, which describes the state {@link Transition}s made by the automaton and indicates whether the terminal state is an accepting state. Deterministic Finite Automata (DFA) are implemented in the finiteautomaton.dfa package; Nondeterministic Finite Automata (NFA) are implemented in the finiteautomaton.nfa package. Both DFAs and NFAs can be loaded from a Rivas Finite Automaton (RFA) file using the {@link FiniteAutomaton.loadFromFile()} method.
